积累本模板

积累本模板

对象：Grace 的孩子 用途：每次订正错题后，花 3-5 分钟手写填入 配用文档：《题目订正与积累工作流.md》《每题阅读引导模板.md》

〇、为什么用手写积累本

手写比打字记得牢——这是认知科学的结论。写字时手指的运动、眼睛跟随笔画的移动、每一笔的用力，都在给大脑提供多通道的输入。打字只有手指的键盘记忆。

更重要的是——手写积累本让孩子看到自己的"知识资产"在变厚。一本写满的方格本，比一个装满文件的文件夹，更能让孩子感受到"我在进步"。

📒 建议：买一本 B5 大小的方格本（约 100 页），专门做这个积累本。不用贵，MUJI 或国誉都可以。封面上写："文言文积累本 · 我的知识武器库"。

一、积累本的页面结构

每页分 7 栏（横式排版，推荐 B5 方格本）

┌──────┬────────┬────────────────┬──────────────┬────────────┬──────────┬──────┐
│ 日期  │  出处  │    高频词       │    句式       │ 文化常识   │  我的错   │ 复习  │
├──────┼────────┼────────────────┼──────────────┼────────────┼──────────┼──────┤
│      │        │                │              │            │          │ □□□ │
│      │        │                │              │            │          │ □□□ │
│      │        │                │              │            │          │ □□□ │
└──────┴────────┴────────────────┴──────────────┴────────────┴──────────┴──────┘

栏目说明：

栏名 宽度占比 写什么 示例
日期 8% 哪一天订正的 6.29
出处 12% 哪篇传记 / 哪道题 王蕴传·题1
高频词 30% 这一句里学到的所有实词（含释义 + 出自哪句） 白：禀告（"辄连状白之"）
句式 20% 主谓宾拆解（用 / 分割） 晖 / 刚 / 于为吏
文化常识 15% 官职、年号、谥号、称谓、制度等 "白"是下对上的报告用语
我的错 10% 我当时怎么想的（只写一句话） 以为"白"是白色
复习 5% 每次复习后画一个勾 □□□ → ✓✓□

三、积累本的使用方法

每天订正完就写（5 分钟）

订正完一道错题，立刻翻开积累本，把这一道题里学到的东西写进去。不要留到明天写——过了今晚就忘了大半。

每周复盘一次（翻看本周所有页面）

周末花 10 分钟，把本周写的所有页面翻一遍： - 看着"高频词"栏，口头说出每个词的意思 - 看着"句式"栏，口头读一遍拆好的句子 - 在"复习"栏里画一个勾 ✓

每道题复习 3 次才算"消化"

第几次 什么时候 做什么
第 1 次 订正当日 写完积累本
第 2 次 本周复盘日 翻看 + 口头复述
第 3 次 下周复盘日 翻看上周页面 + 只看"我的错"栏 → 是否还记得

3 次复习 = 将一个知识点从"短期记忆"转入"长期记忆"。

怎么快速找到某个词

积累本按日期排，不是按主题排。如果孩子想找某个词（比如"我上次见过'属'，在哪儿来着？"）：

方法 1：用标签贴纸。在方格本右侧边缘贴彩色标签——红色=高频词集中页、蓝色=句式集中页、绿色=文化常识集中页。

方法 2：在封底留 3 页做索引页。格式：

高频词索引（按拼音首字母）：
白（禀告）→ 6.29, 7.5
固（坚决）→ 6.29
属（嘱托）→ 6.30, 7.8
……

四、积累本和错题本的区别（重要）

很多家长会让孩子做"错题本"，那个是用来考前复习的。积累本不同——它是用来日常学习的。

错题本 积累本
目的 记录做错的题，考试前翻看 把每道题的知识点分类储存
组织方式 按题号排列 按"高频词/句式/文化常识"分类
记录内容 题号 + 我的答案 + 正确答​案 这一句里所有学到的新东西
使用场景 大考前翻 每天订正后写，每周翻
感觉 "我错了这么多"（挫败感） "我积累了这么多"（成就感）
入门阶段 ❌ 不推荐（太打击） ✅ 强烈推荐（正反馈）

入门阶段只用积累本，不用错题本。 错题本等到入门阶段结束后（能稳定做对 50%+ 的题）再开始做。

五、积累本进阶（入门阶段结束后升级）

入门阶段结束后，积累本可以逐步升级：

阶段 变化
入门阶段（现在） 7 栏（日期/出处/高频词/句式/文化常识/我的错/复习）
进阶阶段 增加一栏"断句标志"（记录每次断句题用到的标志词：也/矣/乎/哉/以/而…）
冲刺阶段 增加一栏"真题出处"（这个词在哪些年份的高考题中出现过）

六、给孩子的话（Grace 可以读给孩子听）

"这本本子不是作业本，也不是改错本。它是你自己的'知识武器库'。"

"每次订正完一道题，你把学到的东西写进去——一个字、一个句式、一个古代的冷知识——都是你的武器。"

"现在是 6 月底。你想想，到 8 月底，这本本子写满了 30 页，里面存着 100 多个你亲手写的词——那时候你再拿起一篇文言文，那些你写过的词会从纸上跳出来。"

"高中的文言文阅读，说到底就是在认字。你早认完 200 个，高考文言文对你来说就不再是'看天书'，而是'这不是我积累本里那个词吗？'"

"每天订正完花 5 分钟写一下，就像打游戏存档——存了就不会丢。"
