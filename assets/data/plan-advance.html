12 周进阶方案

文言文辅导方案 · 人物传记类（上编）

对象：Grace 的孩子 当前水平：约 55-65%（高考要求基准） 目标：3 个月内提升至 75-85% 核心痛点：断句 > 翻译 > 实词积累

一、总方针

四步循环法：精读 → 断句 → 翻译 → 复盘

每篇传记：先通读理解 → 逐句断句 → 逐句翻译 → 对照答案查漏 → 积累高频词

每周完成 2 篇人物传记精读（约 2-3 小时/周）

二、针对人物传记类的专项学习方法

2.1 高频词系统积累（最紧迫）

人物传记类文言文有高度固定的"词汇场"，一旦建立系统认知，阅读效率倍增。

2.2 传记文体特征认知

人物传记类文言文有固定的"叙事模板"，识别后可大幅提升阅读速度：

【开头】姓名 + 字 + 籍贯 + 家世 + 早年特点
【中段1】入仕经过（举/辟/征/拜/除……）
【中段2】主要事迹（1-3 件），常含对话/劝谏
【中段3】为人品性（正面/侧面描写）
【结尾】结局（卒/薨/病卒）+ 谥号 + 追赠
【论赞】作者的"论曰""赞曰"评价

📋 练习方法：每读一篇传记，先画"叙事结构图"，标注各段落在模板中的位置。

三、错因对应的练习建议

3.1 断句（最弱项，优先攻克）

诊断：孩子对文言句子结构没有"肌肉记忆"，凭感觉断，而非凭语法。

方法： 1. 找动词法：每句话先找谓语动词 → 往前找人名/主语 → 往后找宾语 → 由此确定句子边界 2. 虚词定位法： - 句首发语词（夫、盖、且、若夫）前断开 - 句末语气词（也、矣、乎、焉、耳、哉）后断开 - 连词（而、则、乃、且、遂、因）前常断 3. 句式标志法： - "……者，……也" → 判断句，中间必断 - "为……所……""见……于……" → 被动句，注意边界 4. 每日练习：每天 3 句断句（找真题/模拟题），用"/"标注，对照答案，分析每处断句的理由

推荐训练材料： - 《五年高考三年模拟》文言文断句专项 - 任意《史记》《汉书》人物传记段落（无标点版）

3.2 翻译（第二大弱项）

诊断：翻译方法不规范，关键词会漏掉，句式转换不准确。

方法： 1. 三遍翻译法： - 第一遍：逐字直译（每个字都翻，不许跳过） - 第二遍：调整语序（如倒装句还原、省略句补足） - 第三遍：通顺润色（使之像现代汉语）

  1. 翻译六查（译完必查）： - 实词是否逐字落实？ - 虚词是否体现？（之乎者也而于以焉） - 句式是否转化？（被动→"被"字句，判断→"是"字句，倒装→还原） - 省略是否补足？（主语/宾语省略要补） - 专名是否保留？（人名/地名/官名/年号不译） - 是否通顺？（默读一遍）

  2. 易错句式专项： - 被动句：为……所…… / 见……于…… / 被 / 受 - 倒装句：宾语前置（何……之有 / 唯……是……） - 省略句（最常见，主语承前省略）

3.3 实词积累

诊断：一词多义的分辨力弱，古今异义意识不足。

方法： 1. 语境推断训练：遇到生词，先不查词典，尝试从上下文推断 - 看句子结构判断词性 - 看前后文推断语义 - 用已学义项排除法

  1. 词卡法：正面写实词+出处，背面写释义+例句，每天 5 分钟速过

  2. 错题词集：每错一个实词，记入专属错题本，标注错因（"我误以为……"）

3.4 特殊句式识别

诊断：能读懂大意，但遇到被动句、倒装句容易"混过去"，翻译时体现不出来。

方法： 1. 句式标志词清单： - 判断句：……者……也 / 乃 / 即 / 则 / 为 / 是 / 非 - 被动句：为……所…… / 见……于…… / 被 / 于 - 倒装句：宾语前置（疑问代词作宾语 / 否定句中代词作宾语） - 定语后置：……之……者 / 中心词+定语+者

  1. 标注训练：读文言文时，拿笔圈出所有句式标志词，养成条件反射

3.5 人物传记专有词

诊断：不熟悉官职、谥号、年号、称谓等文化常识。

方法： 1. 制作"官职动词一张表"（见上文 2.1），贴书桌前随时看 2. 每次遇到不认识的官职名，查一下大致对应什么级别（中央/地方/军事） 3. 了解常见谥号含义（文/武/景/宣/穆/昭……）

四、下一阶段训练方向

4.1 推荐书目

拓展（持续） 《世说新语》（短篇，趣味性强） ⭐⭐ 培养语感 + 文化常识

五、关键提醒

  1. 断句是突破口：断句最能体现文言文理解水平，也是提分最快的项目。前两周集中攻克断句，可以快速建立信心。

  2. 不要死记硬背：文言文不是靠背的，是靠"读"出来的。每天 15 分钟朗读（出声读！），培养文言文语感，比背 100 个实词更有效。

  3. 错题本是命：每道错题记录三要素——①我当时的答案 ②正确答案 ③我错在哪里（写一句话总结）。每周翻一次。

  4. 人物传记是"舒适区"：相比于论说文、游记、序跋，人物传记的叙事性强、模式固定，最适合入门训练。先在此类上打好基础，再拓展到其他文体。

  5. 建议 Grace 参与：如果 Grace 能每周陪孩子复盘一次错题（15 分钟即可），效果会好很多。不是辅导，就是一起看看"这周哪里进步了"。
