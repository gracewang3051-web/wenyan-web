用途：未来知识点积累 + 复习参考 来源：语文练习题-第一章人物传记类（上编） 配套播客：播客版-第12篇-逊斋记.md

一、实词

1. 逊（xùn）⚠️ 高考高频

  • 本文义：谦下
  • 其他常见义：① 出让（逊位）；② 差、不如（"稍逊一筹"）
  • 高考常考义项：谦下、辞让
  • 易混：让（ràng，侧重于行为上的礼让，与"逊"的退一步策略不同）

2. 信（xìn）⚠️ 高考高频

  • 本文义：使……成（"信以成之"——用真诚来完成它）
  • 其他常见义：① 信任；② 信物；③ 确实、果真；④ 任意（信步）
  • 高考常考义项：信任 / 确实 / 诚信

3. 胜（shēng）

  • 本文义：尽（"可胜用哉"——能用得完吗）
  • 多音辨析：shēng（尽）vs shèng（胜利）
  • 高考常考例句：孟子"谷不可胜食也"；《鸿门宴》"刑人如恐不胜"
  • 翻译口诀：后面接数量或范围概念→读 shēng，"尽"义

4. 约（yuē）⚠️

  • 本文义：要领、关键（"守约之道"——抓住关键的道理）
  • 其他常见义：① 约定；② 简约；③ 约束
  • 高考常考义项：要领（守约而施博）、约定
  • 易混："约"在先秦散文中常做"要领"，不同于现代汉语"约定"

5. 旨（zhǐ）

  • 本文义：美、好（"旨哉言乎"——这话说得好啊）
  • 其他常见义：① 主旨、意图；② 皇帝的诏书（圣旨）
  • 高考常考义项：美好 / 主旨

6. 婉（wǎn）

  • 本文义：温和（"辞逊而弗婉"——言语谦逊却不够温和）
  • 扩展：婉辞 = 温和的措辞
  • 高考常考义项：温和、委婉

7. 干（gàn）

  • 本文义：树干、根本（"义之在人，犹物之有干也"）
  • 其他常见义：① 做事（才干）；② 涉及（干系）；③ 盾牌（干戈）
  • 高考常考义项：根本 / 才干

8. 直遂（zhí suì）

  • 本文义：直接达成（"义有不可直遂者"）
  • 拆解：直 = 直接；遂 = 达成、成功
  • 高考提示："遂"常考"成功/达成"义，如"功成名遂"

9. 跬步（kuǐ bù）

  • 本文义：半步、小步（"积跬步，而至千里"）
  • 出处：《荀子·劝学》"不积跬步，无以至千里"
  • 高考提示：常与"千里"形成对比，用于论述积累的重要性

10. 赍（jī）

  • 本文义：凭借、携带（"何所赍而行哉"——靠什么往前走呢）
  • 其他常见义：① 送给、给予（赍送）；② 怀着（赍志而殁）
  • 高考提示：字形从"貝"，与财物有关；引申为"持有/凭借"

11. 箧（qiè）

  • 本文义：书箱（"入学鼓箧"）
  • 高考提示：常见搭配"书箧""鼓箧"；注意与"筐"区分

12. 孜孜（zī zī）

  • 本文义：勤奋不倦（"学殖之务，尤孜孜焉"）
  • 常见搭配：孜孜不倦
  • 高考提示：叠词表状态，翻译为"勤奋的样子"

13. 取则（qǔ zé）

  • 本文义：取法、效法（"莫不于此取则焉"）
  • 拆解：取 = 拿来；则 = 法则、标准
  • 高考提示："则"作名词表"法则/标准"是常见考点

二、虚词

1. 之 ⚠️

  • 本文用法：助词 / 代词
  • 代词：代事物（"有行之者"——做这件事的人）
  • 助词：的（"入门第一义"之后省略的"之"）；取消句子独立性（"人之……"）
  • 高考提示："之"作代词时要准确补出所指代对象

2. 而 ⚠️

  • 本文用法：
  • 顺承："而可终身行之者乎"（而且）
  • 转折："虽然"（然而）
  • 修饰："逊以出之"中的"以"表方式→"而逊以出之"
  • 高考提示：注意判断"而"的六种关系（顺承/转折/并列/修饰/递进/因果）

3. 焉

  • 本文用法：
  • 兼词（于之）："取则焉" = 取则于是（从这里取法）
  • 语气词："孜孜焉"（……的样子）
  • 高考提示："焉"作兼词是高频考点，常见于句末

4. 乎 ⚠️

  • 本文用法：疑问语气词（"有一言，而可终身行之者乎？"）
  • 其他用法：介词（同"于"）；感叹语气
  • 高考提示：句末"乎"大多是疑问语气，译为"吗/呢"

5. 盖

  • 本文用法：发语词，引起议论（"盖人之一生"——大概/一般而言）
  • 高考提示：句首"盖"不译，或译为"大概"

6. 苟

  • 本文用法：如果、假如（"苟有得于一言而合于道"）
  • 高考提示："苟"在文言中几乎都译为"如果"

7. 虽

  • 本文用法：即使（"故虽一言也"——所以即使只是一个字）
  • 高考提示：注意区分"虽"（即使）和"虽然"（虽然这样），现代汉语"虽然"在古文中是"虽……然……"

8. 然则

  • 本文用法：既然这样，那么……（"然则守约之道，顾不至当欤？"）
  • 高考提示："然则"是固定搭配，不要拆开翻译

9. 欤（yú）

  • 本文用法：疑问语气词（"顾不至当欤？"——难道不是最恰当的吗）
  • 高考提示：同"与"，句末疑问语气

三、特殊句式 ⚠️

1. 判断句：「……者，……也」

  • 例句：「逊也者，吾党之士入门第一义，出门第一程也。」
  • 结构：A 者，B 也 → A 是 B
  • 翻译要点：译为"是"
  • 高考提示：判断句是文言文句式基础考点，注意"者…也"标记

2. 倒装句（状语后置）

  • 例句：「有得于一言而合于道」→ 于一言有得 / 于道合
  • 翻译要点：把"于……"提前到动词前翻译

3. 反问句

  • 例句：「可胜用哉！」（能用得完吗！= 用不完）
  • 「顾不至当欤？」（难道不是最恰当的吗？= 是最恰当的）
  • 翻译要点：译出反问语气

4. 宾语前置

  • 例句：「何所循而进，何所赍而行哉！」
  • 结构："何所 V"→"所 V 者为何"——遵循什么（向前进），凭借什么（向前走）
  • 高考提示："何所……"是宾语前置的一种形式

5. 双重否定

  • 例句：「又不可不信以成之也」（不可不 = 必须）
  • 翻译要点："不可不" = "必须/一定要"

6. 省略句

  • 例句：「终始惟一言也」→ 省略主语"他（司马光）"
  • 高考提示：翻译时补出省略的主语

四、翻译重点 ⚠️

1. 关键词翻译

词 文中义 翻译
逊 谦下 谦虚
胜（shēng） 尽 用尽、用完
守约 抓住要领 守住最关键的道理
信以成之 用真诚来完成 用真诚来成就它
逊以出之 以谦虚的方式表达 用谦虚的态度把道理讲出来

出门第一程 处世第一段路 走向社会的第一步

2. 句式转化要点

古文句式 转化方法 译文示例
「……者，……也」 加"是" 逊，是我们读书人的入门第一要义
「非所谓……者乎」 译为"这不就是……吗" 这不就是可以终身奉行的字吗
「虽……也」 "即使……也" 即使只是一个字
「不可不……」 "必须/一定要" 又一定要用真诚来成就它
「何所V」 "V什么" 靠什么向前进

3. 常考翻译句 ⚠️

① 「逊以出之，又不可不信以成之也。」 - 采分点：逊（以谦虚的方式）、以（用……来……）、信（真诚） - 参考译文：用谦虚的方式把道理表达出来，又一定要用真诚来成就它。

② 「君实之于实，自为儿时至于相天下，终始惟一言也。」 - 采分点：之（对于）、至于（一直到）、终始（从头到尾） - 参考译文：司马光对于"实"这个字，从小孩的时候一直到做了宰相，从头到尾就这一个字。

③ 「逊也者，吾党之士入门第一义，出门第一程也。」 - 采分点：吾党（我们这些）、第一义（首要原则）、程（路程） - 参考译文：谦逊，是我们这些读书人进学的第一要义，也是处世的第一段路。

六、文化常识

播客版板块六已详讲：司马光"守实"、元朝士大夫的"逊"、升堂入室。此处仅列清单要点。

1. 司马光字君实

  • 北宋史学家，主持编修《资治通鉴》
  • "实"为其一生信仰，"君实"即"以实为君（主宰）"

2. 四书引用

  • 本文引《礼记》（入学鼓箧）、《尚书·说命》（惟学逊志）、《论语》（义以为质，逊以出之）、《孟子》（孟施舍之守气，不如曾子之守约）

3. 入学鼓箧

  • 古代入学仪式：击鼓（振作精神）、打开书箱（取出经书）→ 以谦逊为先

4. 升堂入室

  • 出自《论语·先进》孔子评子路
  • 升堂 = 学问已入门（前厅）、入室 = 达到精深（内室）
  • 常见考点：比喻学习的不同阶段

5. 孟施舍与曾子

  • 孟施舍：古代勇者，以"养气"著称
  • 曾子：孔子弟子曾参，以"守约"（抓住要害）著称
  • 出自《孟子·公孙丑上》

七、文意理解

1. 全文主旨

  • 表面：论"逊"（谦下）是一个人可终身奉行的字
  • 深层：欧阳玄在元朝朝廷的生存策略——"逊"不是道德说教，是在力量不对等时的清醒选择

2. 文章结构

  • 第一部分：设问→司马光守"实"的例子→建立"守约之道"框架
  • 第二部分：吴礼逊登场→从《礼》《书》《论语》论证"逊"（入门第一义 + 出门第一程）
  • 第三部分："逊"需要"信"来成全→引用孟子收尾

3. 关键论证逻辑

  • 司马光守"实"→证明"守约之道"可行→吴礼逊守"逊"亦可→但需"信"加持
  • "入门第一义"（对内谦虚学习） + "出门第一程"（对外温和处世）= 完整的"逊"

4. 隐含信息

  • 吴礼逊名"礼逊"→ 作者从名字出发，把个人名号升华为哲学命题
  • 欧阳玄"南人"身份→在元朝需要"逊"作为生存策略

八、写作手法

1. 引经据典 ⚠️

  • 引《礼记》《尚书·说命》《论语》《孟子》四部儒家经典
  • 引司马光为实例
  • 作用：增强论证权威性 + 从不同角度论证"逊"

2. 借题发挥、以小见大

  • 从吴礼逊的名字出发，将"逊"扩展为一个可终身奉行的哲学命题
  • "记"体文章通例：从具体事物引申到普遍道理

3. 先立其大者（论点前置）

  • 开篇不直接谈"逊"，先建立"守约之道"的框架→再放入"逊"→分量翻倍
  • 议论文常见策略：先确立论证框架，再填充具体内容

4. 层层递进

  • 设问→举例→具体论证（入门/出门）→补充条件（信）→引用圣人收尾
  • 每一层都在前一层基础上加深

5. 对偶句式

  • 「入门第一义，出门第一程」
  • 「婉辞足以达意，愉色足以致和」
  • 「升堂而望入室」「积跬步而至千里」
  • 作用：增强节奏感 + 并列论证 + 便于记忆

6. 比喻论证

  • 「义之在人，犹物之有干也」—— 义是人的树干（根本）
  • 「升堂望入室」—— 学习如登堂入室
  • 「积跬步至千里」—— 处世如行路

7. 首尾呼应

  • 开头："有一言而可终身行之者乎" + "司马光守实"
  • 结尾："信以成之，则如君实之守其实而后可也"——再次回到司马光
